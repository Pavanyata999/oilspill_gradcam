💡 SEGMENTATION COMPARISON EXPLAINED
================================================================================

WITHOUT Post-Processing:
  • Raw neural network predictions
  • Threshold at 0.5
  • Faster but noisier
  • More false positives

WITH Post-Processing:
  • Adaptive thresholding (model-specific)
  • Morphological operations (cleanup)
  • Small object removal
  • Cleaner masks, fewer false alarms
  • 5-10% precision improvement

🎓 UNDERSTANDING THE OUTPUTS
================================================================================

ROC Curve:
  • X-axis: False Positive Rate (lower is better)
  • Y-axis: True Positive Rate (higher is better)
  • Curve close to top-left = excellent
  • AUC > 0.9 = excellent performance

Confusion Matrix:
  • Top-left (TN): Correct non-spill predictions
  • Top-right (FP): False alarms ← Want to minimize
  • Bottom-left (FN): Missed spills ← Want to minimize
  • Bottom-right (TP): Correct spill detections

Metrics Chart:
  • Blue bars: Without post-processing
  • Red bars: With post-processing
  • Higher bars = better performance
  • Compare across models
